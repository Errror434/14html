Адаптивна веб-сторінка

Це приклад веб-сторінки з адаптивним дизайном. Використовуються додаткові можливості форматування тексту.

Цитати

Ось приклади використання тегів <q> та <blockquote> для цитат:

Мова - це вогнемет, що його людство гасить сотнями законів. - Марк Твен

Люди, які не читають, мають перевагу: вони можуть розмовляти з людьми, яких немає. - Станлі Уїнтроп

Абревіатури

Ось приклад використання тегу <abbr> для абревіатур:

WWW - це система розподіленої інформації, що доступна через Інтернет.

Адреси

Ось приклад використання тегу <address> для адреси:

1600 Amphitheatre Pkwy, Mountain View, CA 94043, USA

Цитування

Ось приклад використання тегу <cite> для цитування джерела:

Віра в себе - це перша таємниця успіху. - Ральф Уолдо Емерсон

Обернений напрямок тексту

Ось приклад використання тегу <bdo> для оберненого напрямку тексту:

Привіт, світ!
